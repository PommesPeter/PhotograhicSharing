## 用户

### 修改头像

>请求地址：http://172.16.75.73:9192/users/modify_avatar
>
>请求示例：http://172.16.75.73:9192/users/modify_avatar?id=admin
> 
>请求方式：POST
>
>返回格式：UTF8 JSON

#### 请求参数

| 名称  |  类型  | 必填  |    说明    |
| :---: | :----: | :---: | :--------: |
|  id   | string |  是   | 用户的账户 |

#### 返回值

```json
{
    "msg":  "success"
    "data": [
        {
            "url": ""
        }
    ]
}
```

#### 请求参数

| 名称  |  类型  | 必填  |     说明     |
| :---: | :----: | :---: | :----------: |
|  id   | string |  是   |  用户的账户  |
|  url  | string |  是   | 用户的新头像 |

#### 返回值

```json
{
    "msg":  "success"
    "data": []
}
```
